================================================================================
  🛡️  BẢO VỆ BẢN QUYỀN
================================================================================

  File gốc cần giữ:
    ✅  .ai   →  Quan trọng nhất (chứa vector paths, layer, metadata)
    ✅  .psd  →  Nếu làm trên Photoshop (.psb nếu file >2GB)
    ❌  .png  →  Không cần giữ, export lại từ .ai khi cần

  Thứ tự lưu backup:
    1. Máy local  (bản gốc, không bao giờ xóa)
    2. Lark Drive  (100GB miễn phí, nhanh ở VN)
    3. Google Drive  (backup phụ, file quan trọng nhất)

  Bằng chứng bản quyền khi tranh chấp:
    → File .ai gốc  +  Timestamp Lark/Drive  +  Ngày upload Fourthwall
