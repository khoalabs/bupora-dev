	🔐 Bảo mật cho push (create/delete/update) bài viết qua API bằng script
		Google Colab server.ipynb
			1. API Key trong header (không dùng query param)
				Colab gửi:
					Authorization: Bearer YOUR_SECRET_KEY
					Thêm
						X-Admin-Version: 1.0
					Hoặc:
						Timestamp + HMAC (nâng cao hơn nếu bạn muốn hardcore security).
				Server kiểm tra:
					from functools import wraps
					from flask import request, abort
					import os

					API_KEY = os.environ.get("ADMIN_API_KEY")

					def require_api_key(f):
							@wraps(f)
							def decorated(*args, **kwargs):
									auth = request.headers.get("Authorization")
									if not auth or auth != f"Bearer {API_KEY}":
											abort(401)
									return f(*args, **kwargs)
							return decorated
			2. Giới hạn IP qua Nginx (khuyên dùng)
				Nếu Colab có IP cố định (không luôn luôn ổn định),
				ít nhất bạn có thể:
					Cho phép chỉ US
					Hoặc bật rate limit
				Hoặc đơn giản:
					Chỉ mở endpoint khi publish xong → tắt lại.

	🔎 Full-text search tối ưu Postgres
		ALTER TABLE articles ADD COLUMN tsv tsvector;
		UPDATE articles SET tsv = to_tsvector('english', title || ' ' || content);
		CREATE INDEX idx_tsv ON articles USING GIN(tsv);
		
		Trigger auto update:
			CREATE TRIGGER tsv_update BEFORE INSERT OR UPDATE ON articles FOR EACH ROW EXECUTE FUNCTION tsvector_update_trigger(tsv, 'pg_catalog.english', title, content);
		
		Search query:
			SELECT * FROM articles WHERE tsv @@ plainto_tsquery('english', 'golden retriever grooming');
	📄 Phân trang
		Dùng LIMIT / OFFSET.
			SELECT * FROM articles ORDER BY created_at DESC LIMIT 10 OFFSET 20;

✅ API Key (Bearer secret)
✅ Timestamp (chống replay)
✅ HMAC SHA256 (chống giả mạo / sửa body)
✅ X-Admin-Version
✅ So sánh signature bằng compare_digest
✅ Rate limit nhẹ ở Nginx
🎯 Chuẩn giao thức request
	Colab phải gửi các header sau:
		Authorization: Bearer <API_KEY>
		X-Timestamp: <unix_timestamp>
		X-Admin-Version: 1.0
		X-Signature: <hmac_sha256>
		Content-Type: application/json
🔐 Chuẩn tính chữ ký (RẤT QUAN TRỌNG)
	Message để ký phải thống nhất 100%.
	Ta định nghĩa:
		message = HTTP_METHOD + "\n" + REQUEST_PATH + "\n" + timestamp + "\n" + raw_body
	Ví dụ:
		POST
		/api/admin/create
		1710000000
		{"title":"abc"}
	Nếu không cố định format → mismatch rất khó debug.
